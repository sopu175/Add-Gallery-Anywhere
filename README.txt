== Description ==

The **Add Gallery Anywhere with Fancybox** plugin allows you to dynamically add image galleries to any page or post using a simple shortcode. The plugin integrates **Fancybox** with the **HTML plugin** to create interactive and responsive galleries. You can display your images in a beautiful lightbox with captions and alt text.

**Features**:
- Easily add galleries to any page/post using a simple shortcode.
- Supports multiple images, captions, and alt text.
- Fully responsive with **Fancybox** integration for the lightbox.
- Simple to use and easy to customize.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/add-gallery-anywhere` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the [add-gallery-anywhere] shortcode on any page or post to insert your gallery.
4. Add your gallery images in the pluginâ€™s settings or in the gallery post.
5. Use the generated shortcode to display your gallery anywhere on your website.

== Frequently Asked Questions ==

= How do I add a gallery to a page or post? =
Simply use the `[add-gallery-anywhere id="your-gallery-id"]` shortcode on any page or post.

= Does this plugin support lightbox functionality? =
Yes! This plugin uses Fancybox with the HTML plugin to create a lightbox effect for your gallery images.

= How do I change the number of images per row? =
You can configure the number of images shown per row in the plugin's settings or within the gallery post metadata.
